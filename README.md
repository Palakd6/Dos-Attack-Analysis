# 📡 Network Packet Transmission and Analysis using Wireshark

## 📌 Project Overview

This project demonstrates how network packets can be generated using Python scripts and analyzed using Wireshark. The goal is to understand how data travels across a network and how packet sniffing tools can be used for monitoring and learning purposes in a controlled environment.

---

## 🎯 Objectives

* To generate network traffic using Python
* To capture and analyze packets using Wireshark
* To understand basic networking protocols such as TCP/IP and HTTP
* To study packet structure and communication patterns

---

## 🛠️ Tools and Technologies Used

* **Python 3.x**
* **Wireshark** (Network Protocol Analyzer)
* **Requests Library (Python)** *(if used)*
* Operating System: Windows/Linux

---

## 📂 Project Structure

```
├── sender.py          # Script to generate/send packets
├── receiver.py        # Script to receive/process packets (if applicable)
├── README.md          # Project documentation
```

---

## ⚙️ Working of the Project

### 1. Packet Generation (Python Script)

The `sender.py` script is responsible for generating network traffic. It sends packets over the network using Python libraries such as `socket` or `requests`.

### 2. Packet Reception (Optional)

The `receiver.py` script listens for incoming packets and processes them accordingly (if implemented).

### 3. Packet Capture (Wireshark)

Wireshark is used to capture the packets transmitted over the network. Filters can be applied to analyze specific types of traffic such as:

* HTTP
* TCP
* DNS

---

## ▶️ How to Run the Project

### Step 1: Start Wireshark

* Open Wireshark
* Select your active network interface (Wi-Fi/Ethernet)
* Start capturing packets

### Step 2: Run Python Script

```bash
python sender.py
```

*(Run receiver.py if applicable)*

```bash
python receiver.py
```

### Step 3: Analyze Packets

* Observe captured packets in Wireshark
* Apply filters such as:

```
http
tcp
dns
```
